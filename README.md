📸 JB Creative Cinema — Photography & Videography Website

A modern, responsive photography and videography website built to showcase visual work, attract clients, and enable seamless booking experiences.



🧰 Tech Stack

Next.js (App Router)

React

TypeScript

Tailwind CSS

Next/Image Optimization

Responsive Design (Mobile-first)

🎯 Project Purpose

This project was built as a real-world business website for a creative media company to:

Showcase photography and videography work

Provide business information and branding

Enable users to book sessions

Improve online visibility and client engagement

✨ Key Features
📌 Modern UI/UX

Clean, luxury-inspired design

Smooth gradients and dark/light mode support

Fully responsive across all devices

🖼️ Interactive Gallery

Infinite scroll slider

Center-mode focus (highlighted image)

Click-to-expand image modal view

👥 About & Team Section

Business mission, vision, and services

Team member cards with roles and profiles

📊 Animated Stats Section

Scroll-triggered counter (e.g. Happy Clients, Projects)

Enhances user engagement and credibility

📍 Location & SEO

Embedded Google Maps integration

Structured metadata for local SEO

Business discoverability improvements

📅 Booking System (Frontend)

User form for capturing client details

Ready for backend/API email integration

📱 Mobile Navigation

Responsive hamburger menu

Auto-close on interaction for better UX
